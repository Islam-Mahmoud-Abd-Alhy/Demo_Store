# ⚡ Demo Store - Cyberpunk E-Commerce UI

Welcome to **Demo Store**, a modern, responsive e-commerce frontend project designed with a futuristic **Cyberpunk** aesthetic. This project focuses on high-quality UI/UX, smooth animations, and a seamless mobile experience.

---

## 🚀 Features

* **Futuristic UI:** Dark theme with neon accents and glassmorphism effects.
* **Fully Responsive:** Optimized for all screen sizes, featuring a custom-built mobile navigation menu.
* **Interactive Animations:** Powered by **GSAP** for smooth transitions and scroll-triggered effects.
* **Smart Shopping Cart:** Functional logic for adding products and calculating totals (String-to-Number parsing).
* **Modern Notifications:** Styled alerts using **SweetAlert2**.
* **Contact Integration:** Ready-to-use email functionality via **EmailJS**.

---

## 🛠️ Tech Stack

* **HTML5** - Semantic structure.
* **CSS3** - Custom properties (Variables), Flexbox, Grid, and Media Queries.
* **JavaScript (ES6+)** - Logic for the cart and mobile toggle functionality.
* **GSAP** - Professional animations.
* **SweetAlert2** - Beautiful popup dialogs.
* **EmailJS** - Client-side email service.
